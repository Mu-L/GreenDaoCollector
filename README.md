# GreenDaoCollector
Auto-collect daos' info to help db upgrade. 常见的GreenDao迁移做法都比较粗暴，但是人为维护迁移信息容易造成疏漏。

提供两种方案：

* 利用注解处理进行收集
* 魔改插件替换FreeMaker模板进行收集
